### setup .env

1. make a copy of `.env.local.example` named `.env.local`
1. edit `.env.local` and add yr Ethereum node URL!

Example: `ETHEREUM_NODE_URL=https://mainnet.infura.io/v3/YOUR_INFURA_ID`

### run on local:

1. `npm i`
1. `npm run build`
1. `npm run dev`
1. `🙂`
